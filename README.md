# bright-key-mapping for autohotkey

---

1. Move
   - left: CapsLock + j
   - left(word): CapsLock + h
   - right: CapsLock + l
   - right(word): CapsLock + ;
   - up: CapsLock + i
   - down: CapsLock + k
   - home: CapsLock + ,
   - end: CapsLock + .
   - page up: CapsLock + u
   - page down: CapsLock + n
   
---

2. Select
   - Shift + MOVE

---

3. Control combinations
   - Ctrl + f: CapsLock + f
   - Ctrl + a: CapsLock + a
   - Ctrl + s: CapsLock + s
   - Ctrl + c: CapsLock + c
   - Ctrl + v: CapsLock + v
   - Ctrl + x: CapsLock + x
   - Ctrl + y: CapsLock + y
   - Ctrl + z: CapsLock + z

---
